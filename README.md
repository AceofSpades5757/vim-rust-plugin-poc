# Description

Proof of concept: Create a Vim plugin in Rust.

If run correctly, the plugin will set the following in Vim.

`let g:rust_plugin = "Hello Vim!"`

# Usage

## Requirements

* Python
* Cargo

## Installation

Using Vim-Plug

`Plug 'AceofSpades5757/vim-rust-plugin-poc'`
